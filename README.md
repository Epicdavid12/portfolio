# David Glazier — Portfolio

Retro-arcade portfolio site. Static HTML/CSS/JS — no build step required.

## Local

```bash
cd website
python -m http.server 8642
# open http://localhost:8642
```

## Layout

| Path | Purpose |
|------|---------|
| `website/` | Deployable site |
| `docs/` | Change documentation |
| `tools/` | Asset pipeline scripts (images, media) |

Raw source media lives outside the repo (`assets/`, `Site files/`) and is not tracked.
